《点亮星际灯塔》

植物园的能源终于恢复了！梦境工厂的警报解除，最后一步，是安放在明亮云端塔台上的防卫星际灯塔（Lighthouse 🔊）。

这种灯塔拥有极其强大的波普能量，通电后，会向正上方、正下方、正左、正右以及四条斜对角线射出警戒光束（Beam 🔊）。

但是有一条铁律：任何两座灯塔的光束绝对不能互相交叉照射！否则就会发生波普大爆炸！我们需要在网格里安全地放置 4 座灯塔。

像素向导提醒道：“放前两个很简单，但放第三个、第四个的时候，你可能会发现不管放在哪都会爆炸。这就是算法里的死胡同！”

不要灰心！遇到大爆炸锁死时，点击时光魔法——撤销（Undo 🔊），这叫做回溯（Backtrack 🔊）。退回到上一个安全的节点，换一条路，这是解决所有复杂难题的终极奥义！
